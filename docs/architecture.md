# Architecture

The ledger has four layers:

1. `fixtures/process-exception-latency.json` models Camunda process exception lanes.
2. `src/index.ts` scores incident backlog, overdue human tasks, retry loops, latency, SLA breach exposure, and unresolved escalations.
3. `sql/process_exception_latency_contract.sql` describes the warehouse extraction contract.
4. `bpmn/payment-exception-resolution.bpmn` keeps the top-risk process lane tied to a concrete BPMN artifact.

The static site is generated by `scripts/prerender.ts` and served from GitHub Pages.
